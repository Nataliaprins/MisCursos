2. Variables y tipos de datos#

En Python no necesitas declarar el tipo de variable, se asigna automáticamente.

Se puede trabajar con datos de tipo numérico o tipo «cadena de texto»

Tipos Numéricos:

Son los bloques básicos para cualquier cálculo matemático

int (Enteros): Números sin decimales, ya sean positivos o negativos (ej. 5, -10)

float (Punto flotante): Números con decimales (ej. 3.14, 2.0)

complex (Complejos): Se usan en ingeniería y matemáticas avanzadas. Tienen una parte real y una imaginaria, representada por una \(j\) (ej. \(2 + 3j\)).

Tipos de Texto:

Solo hay uno, pero es extremadamente versátil.

str (Strings/Cadenas): Texto encerrado en comillas simples (”…”) o dobles (»…»). Python trata el texto como una secuencia de caracteres.

Tipo Booleano:

bool (Booleano): Solo puede tener dos valores: True o False. Se usa para representar condiciones lógicas (ej. es_mayor = True).

Tipos de Secuencias:

Tipo

¿Es mutable?

Definición

list

Sí

Colección ordenada. Se define con corchetes: [1, "hola", 3.4].

tuple

No

Colección ordenada inmutable. Se define con paréntesis: (1, 2, 3).

range

No

Genera una secuencia de números, común en bucles for.

dict

Sí

Colección de pares clave-valor. Se define con llaves: {"nombre": "Ana", "edad": 25}.

3. Operaciones aritméticas#

Python puede usarse como una calculadora.

[14]:

a = 10
b = 3

print(f"Suma: {a} + {b} = {a + b}")
print(f"Resta: {a} - {b} = {a - b}")
print(f"Multiplicación: {a} * {b} = {a * b}")
print(f"División: {a} / {b} = {a / b:.2f}")
print(f"División entera: {a} // {b} = {a // b}")
print(f"Módulo (residuo): {a} % {b} = {a % b}")
print(f"Potencia: {a} ** {b} = {a ** b}")

Suma: 10 + 3 = 13
Resta: 10 - 3 = 7
Multiplicación: 10 * 3 = 30
División: 10 / 3 = 3.33
División entera: 10 // 3 = 3
Módulo (residuo): 10 % 3 = 1
Potencia: 10 ** 3 = 1000

4. Cadenas de texto (Strings)#

Las cadenas se pueden manipular de muchas formas.

[15]:

saludo = "Hola, bienvenido al curso"

# Métodos básicos de strings
print(saludo.upper())          # Todo en mayúsculas
print(saludo.lower())          # Todo en minúsculas
print(saludo.replace("curso", "taller"))  # Reemplazar texto
print(len(saludo))             # Longitud de la cadena

# Concatenar strings
nombre = "Natalia"
mensaje = saludo + ", " + nombre + "!"
print(mensaje)

# f-strings (formato moderno)
print(f"{saludo}, {nombre}!")

HOLA, BIENVENIDO AL CURSO
hola, bienvenido al curso
Hola, bienvenido al taller
25
Hola, bienvenido al curso, Natalia!
Hola, bienvenido al curso, Natalia!

5. Listas#

Las listas son colecciones ordenadas que pueden contener cualquier tipo de dato.

[16]:

# Crear una lista
frutas = ["manzana", "banana", "cereza", "durazno"]
print(f"Lista: {frutas}")

# Acceder a elementos (el índice empieza en 0)
print(f"Primera fruta: {frutas[0]}")
print(f"Última fruta: {frutas[-1]}")

# Agregar y eliminar elementos
frutas.append("uva")
print(f"Después de agregar: {frutas}")

frutas.remove("banana")
print(f"Después de eliminar: {frutas}")

# Longitud de la lista
print(f"Total de frutas: {len(frutas)}")

# Rebanar (slicing)
print(f"Primeras dos: {frutas[:2]}")

Lista: ['manzana', 'banana', 'cereza', 'durazno']
Primera fruta: manzana
Última fruta: durazno
Después de agregar: ['manzana', 'banana', 'cereza', 'durazno', 'uva']
Después de eliminar: ['manzana', 'cereza', 'durazno', 'uva']
Total de frutas: 4
Primeras dos: ['manzana', 'cereza']

6. Condicionales (if / elif / else)#

Permiten ejecutar código según una condición.

[17]:

nota = 85

if nota >= 90:
    print("Excelente")
elif nota >= 70:
    print("Aprobado")
else:
    print("Reprobado")

# Operadores de comparación
print(f"10 > 5: {10 > 5}")
print(f"10 == 10: {10 == 10}")
print(f"10 != 5: {10 != 5}")

Aprobado
10 > 5: True
10 == 10: True
10 != 5: True

7. Ciclos (Loops)#

Ciclo for — recorre una secuencia#

colores = ["rojo", "verde", "azul"]

Ciclo while — repite mientras una condición sea verdadera#

[19]:

contador = 1
while contador <= 5:
    print(f"Contando: {contador}")
    contador += 1  # Incrementar el contador

print("¡Fin del conteo!")

Contando: 2
Contando: 3
Contando: 4
Contando: 5
¡Fin del conteo!

8. Ejercicio práctico#

Ejercicio: Crea una lista con las notas de 5 estudiantes y calcula el promedio.

Intenta resolverlo en la celda de abajo antes de ver la solución.

[20]:

Solución#

[21]:

# Lista de notas
notas = [4.5, 3.2, 1.8, 2.5, 4.8]

# Calcular el promedio
suma = sum(notas)
cantidad = len(notas)
promedio = suma / cantidad

print(f"Notas: {notas}")
print(f"Suma: {suma}")
print(f"Cantidad de estudiantes: {cantidad}")
print(f"Promedio: {promedio:.2f}")

# Determinar si el grupo aprobó
if promedio >= 70:
    print("El grupo aprobó en promedio")
else:
    print("El grupo reprobó en promedio")
