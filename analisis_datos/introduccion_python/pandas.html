Práctica 2: Lectura y manipulación de datos con Pandas#

En esta práctica aprenderás los conceptos básicos de Pandas, la librería más utilizada en Python para el análisis de datos.

Temas:

  • ¿Qué es Pandas?

  • Series y DataFrames

  • Lectura de datos desde archivos

  • Exploración de datos

  • Selección y filtrado

  • Operaciones básicas

  • Ejercicio práctico

1. ¿Qué es Pandas?#

Pandas es una librería de Python diseñada para la manipulación y el análisis de datos estructurados (tablas, hojas de cálculo, bases de datos, etc.).

Sus dos estructuras principales son:

Estructura

Descripción

Series

Una columna de datos (como un vector o una lista con etiquetas).

DataFrame

Una tabla completa con filas y columnas (como una hoja de Excel).

Para usarla, primero hay que importarla:

[2]:

import pandas as pd  # "pd" es la convención estándar
print(f"Versión de Pandas: {pd.__version__}")

Versión de Pandas: 3.0.1

2. Series y DataFrames#

Series#

Una Series es como una lista con índices (etiquetas).

[3]:

# Crear una Serie desde una lista
notas = pd.Series([85, 92, 78, 95, 88], name="Notas")
print(notas)
print(f"\nTipo: {type(notas)}")

# Crear una Serie con índices personalizados
notas_estudiantes = pd.Series(
    [85, 92, 78, 95, 88],
    index=["Ana", "Luis", "María", "Carlos", "Sofía"],
    name="Notas"
)
print("\nNotas con nombres:")
print(notas_estudiantes)

# Acceder a un valor por su índice
print(f"\nNota de Ana: {notas_estudiantes['Ana']}")

DataFrame#

Un DataFrame es una tabla con filas y columnas. Se puede crear desde un diccionario.

[4]:

# Crear un DataFrame desde un diccionario
datos = {
    "Nombre": ["Ana", "Luis", "María", "Carlos", "Sofía"],
    "Edad": [21, 23, 22, 24, 20],
    "Nota": [85, 92, 78, 95, 88],
    "Ciudad": ["Bogotá", "Medellín", "Cali", "Bogotá", "Medellín"]
}

df = pd.DataFrame(datos)
print("DataFrame:")
df

3. Lectura de datos desde archivos#

Pandas puede leer datos de múltiples formatos:

Archivos JSON

pd.read_sql()

Para este ejemplo, guardaremos nuestro DataFrame como CSV y luego lo leeremos.

df.to_csv("estudiantes.csv", index=False)

df_leido = pd.read_csv("estudiantes.csv")

Seleccionar filas con iloc y loc#

  • iloc → selección por posición (índice numérico)

  • loc → selección por etiqueta (nombre del índice)

[9]:

# Seleccionar la primera fila por posición
print("Primera fila (iloc):")
print(df.iloc[0])

# Seleccionar filas 1 a 3
print("\nFilas 1 a 3:")
print(df.iloc[1:4])

# Seleccionar fila y columna específica
print(f"\nNombre de la fila 2: {df.iloc[2, 0]}")
print(f"Nota de la fila 2: {df.iloc[2, 2]}")

Filtrar datos con condiciones#

Puedes filtrar filas que cumplan una condición lógica.

[10]:

# Estudiantes con nota mayor o igual a 90
print("Estudiantes con nota >= 90:")
aprobados = df[df["Nota"] >= 90]
print(aprobados)

# Estudiantes de Bogotá
print("\nEstudiantes de Bogotá:")
bogota = df[df["Ciudad"] == "Bogotá"]
print(bogota)

# Combinar condiciones: nota >= 85 Y edad < 23
print("\nNota >= 85 y edad < 23:")
filtro = df[(df["Nota"] >= 85) & (df["Edad"] < 23)]
print(filtro)

Agrupar datos con groupby#

Permite agrupar filas por una columna y calcular estadísticas por grupo.

[13]:

# Promedio de nota por ciudad
print("Promedio de nota por ciudad:")
print(df.groupby("Ciudad")["Nota"].mean())

# Contar estudiantes por ciudad
print("\nEstudiantes por ciudad:")
print(df.groupby("Ciudad")["Nombre"].count())

# Múltiples estadísticas por grupo
print("\nEstadísticas de nota por ciudad:")
print(df.groupby("Ciudad")["Nota"].agg(["mean", "min", "max"]))

7. Ejercicio práctico#

Ejercicio: Dado el siguiente DataFrame de ventas, resuelve las preguntas:

  1. ¿Cuántas filas y columnas tiene el DataFrame?

  2. Filtra las ventas mayores a $500

  3. ¿Cuál es el total de ventas por producto?

  4. Ordena el DataFrame por monto de venta de mayor a menor

[14]:

# Datos de ejemplo
ventas = pd.DataFrame({
    "Producto": ["Laptop", "Mouse", "Teclado", "Monitor", "Laptop", "Mouse", "Monitor"],
    "Cantidad": [2, 10, 5, 3, 1, 8, 2],
    "Precio_unitario": [800, 25, 45, 350, 800, 25, 350],
    "Vendedor": ["Pedro", "Ana", "Ana", "Pedro", "Ana", "Pedro", "Ana"]
})

# Agregar columna de venta total
ventas["Venta_total"] = ventas["Cantidad"] * ventas["Precio_unitario"]
